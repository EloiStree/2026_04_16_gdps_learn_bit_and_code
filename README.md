# Learn Bit and Code

> Show how multiple add-ons can be used together to build an IoT-based game for learning coding with the Micro:Bit.

**The idea behind this show case game:**
A scenario is introduced, with several sensors already connected to the Micro:bit.
Your task is to solve the puzzle by writing code, as if you were programming directly on the Micro:Bit.

GDPS stand for Godot Package(s), meaning that I use several packages to make run this one:
- [Micro:Bit](https://github.com/EloiStree/2026_03_20_gdp_hello_micro_bit): To have a moddable Micro Bit
- [Skid Car Toy](https://github.com/EloiStree/2025_10_19_gdp_kid_toy_skid_steering): To have movable two wheel car
- [Mecanum](https://github.com/EloiStree/2026_04_13_gdp_toy_mecanum_robot_code): To have a movable all direction car
- [Learn Sensor](https://github.com/EloiStree/2026_04_11_gdp_learn_with_sensors): To have simulated Server, Led and sensor.
- [Pins](https://github.com/EloiStree/2026_04_11_gdp_pins): To have an abstract communication layer between the game and the user code
- [Nerf](https://github.com/EloiStree/2026_04_12_gdp_nerf_dart): Allow to fire Nerf dart for fun mini game

**For young game developers reading this challenge:**      
> The goal of this training is to guide you in creating simple sensors or interactive objects with multiple pins     
> Example: true/false or values ranging from 0 to 1.        
> These should help a children learn basic concepts of electricity or mechanics,   
> while also introducing game developers to key knowedge of the Godot Engine.        
> _It must be controllable by the pins of a Micro:bit._   
> Good luck, may the code be with you._     

------------

**Next steps:**

* Turn an LED on and off
* Change the color of an LED
* Control a motor (on/off and reverse direction)
* Rotate a servo motor

**Note:**
The goal is not to build a full electricity simulator, but rather:

* to provide an introduction to basic electronics and mechanics using Micro:Bit, aimed at younger students
* to serve as documentation for using the Godot Engine, particularly for game developers
